Spectrum RayTrace(Ray r, Covariance c){

   // Check intersection
   hit = Intersect(scene, r);
   if(! hit) return Black;

   // Evaluate covariance at the surface
   c.Travel(hit);
   c.Projection(hit);

   // Interaction with material
   Ray new_r = SampleBRDF(hit);

   // Compute reflected covariance
   c.Reflection(hit);
   c.InverseProjection(hit);

   return RayTrace(new_r, c);
}

Adaptive Density Estimation


  • Used for Photon Mapping
    • Balance bias and variance using prediction

  • Mathematical insight:
    • Density estimation blurs the ground truth
    • Low variation of GT in the kernel means low bias
    • Theoretically: the $ \mbox{bias} \propto \mbox{Hessian} $
